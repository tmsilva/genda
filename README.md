# 📖 Sobre

O **Genda** é uma plataforma de agendamentos desenvolvida para profissionais autônomos que desejam organizar sua rotina de forma simples, rápida e inteligente.

Mais do que uma agenda, o sistema centraliza clientes, serviços, estoque, agenda e financeiro em um único lugar, proporcionando uma experiência moderna.

Desenvolvido com foco em:

- 🚀 Performance
- 🎨 Interface moderna
- 📱 Responsividade
- 🔒 Segurança
- ⚡ Produtividade

- Cancelamento

## Rodar Localmente

**Prerequisitos:**  Node.js


1. Instale as dependências:
   `npm install`
2. Insira a `GEMINI_API_KEY` no arquivo [.env.local](.env.local) com sua Gemini API key
3. Rode a aplicação:
   `npm run dev`
